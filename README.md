# Final_Project - Lucas Britten
CMPSC 132 Final Project: Guessing Game

# Description
Terminal based python game where the player guesses random numbers with feedback after each guess. The player has a limited number of guesses and different number ranges to guess from based on the difficulty he or she chooses: easy, medium, or hard. At the easy difficulty, the player has 10 guesses to get the number between 1 and 50. At medium, 7 chances for a number between 1 and 100. At hard, 5 chances between 1 to 200.

# How to Play
Run this repository here or clone it.
Run 'number_guessing_game.py' in your own terminal.
Follow the on-screen prompts:
  Choose difficulty
  Guess first number between indicated numbers
  Guess according to the feedback given from the game until the number is guessed or the number of     guesses runs out.
  Choose to play again or not.
